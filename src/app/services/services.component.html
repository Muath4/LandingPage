خدمات البوابة المكانية

  • البحث فى الطبقات الرئيسية والبلديات والأحياء وقطع الأراضي والمخططات والطرق والشوارع سواء على مستوى المدينة أو المحافظات
  • عرض إصدارات للصور الفضائية
  • الاستعلام عن بيانات قطعة الارض
  • رسم خطوط وأشكال وتصديرها إلى Shape file
  • إمكانية إستيراد وتصدير طبقة جديدة في صورة (GIS) Shape file
  • إمكانية طباعة الخريطة
  • البحث عن الأنشطة والخدمات فى نطاق محدد
